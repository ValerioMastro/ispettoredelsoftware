# Demo: Motore di analisi (Swing)

Piccolo prototype Java Swing che implementa una versione minima del diagramma `analisi.puml`.

Prerequisiti:
- JDK 11+
- Maven

Compilare ed eseguire:

```bash
cd progetto
mvn compile exec:java
```

Uso:
- Scegli una directory di sorgenti Java; premi `Run Analysis`.
- Cambia le soglie nella UI per adattare le regole.
